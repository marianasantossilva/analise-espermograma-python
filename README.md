**Professor:** Pedro Torres

## Projeto
Projeto desenvolvido como atividade final da disciplina **Introdução à Programação para Pesquisa Biomédica**.
## Objetivo
Realizar uma análise exploratória de um conjunto de dados sintético de espermogramas utilizando Python.

## Ferramentas utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
- GitHub

## Pipeline do projeto
1. Importação das bibliotecas
2. Leitura da base de dados
3. Padronização dos nomes das colunas
4. Exploração inicial dos dados
5. Estatísticas descritivas
6. Classificação da morfologia espermática
7. Construção de gráficos
8. Interpretação dos resultados
